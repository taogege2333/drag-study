# 安装使用步骤

```shell
# 全局安装pnpm
npm install -g pnpm

# 安装依赖
pnpm install

# 启动
pnpm dev

# 构建
pnpm build

# 使用eslint、stylelint、prettier检测并格式化代码
pnpm lint
```

# git提交步骤

```shell
# 先手动执行lint检查并修复代码，否则提交时校验不通过，会被存储到stash中
pnpm lint

# 将修改的文件添加到暂存区
git add .

# 提交，之后根据提示配置message
git commit
```
